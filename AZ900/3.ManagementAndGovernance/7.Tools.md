# Azure Tools
## Azure Portal
- The Azure Portal is a web-based application that provides a unified interface for managing Azure resources. Key features include:
  - **Resource Management**: Create, configure, and manage Azure resources through a user-friendly interface.
  - **Dashboard Customization**: Personalize your dashboard to display the most relevant information and resources.
  - **Role-Based Access Control (RBAC)**: Manage user access and permissions for Azure resources.

## Azure CLI
- The Azure Command-Line Interface (CLI) is a cross-platform tool that allows you to manage Azure resources from the command line. Key features include:
  - **Scripting and Automation**: Use scripts to automate repetitive tasks and manage resources at scale.
  - **Cross-Platform Support**: Run the Azure CLI on Windows, macOS, and Linux. It cannot run any other commands outside of Azure management.

## Azure PowerShell
- Azure PowerShell is a set of cmdlets for managing Azure resources directly from the PowerShell command line. Key features include:
  - **Integration with PowerShell**: Leverage existing PowerShell skills and scripts to manage Azure resources.
  - **Automation and Scripting**: Use PowerShell scripts to automate complex tasks and workflows.

## Azure Cloud Shell
- Azure Cloud Shell is an online-based shell provided by Microsoft Azure. Key features include:
  - **Browser-Based Access**: Access a command-line interface directly from the Azure Portal without any local installation.
  - **Pre-Configured Environment**: Comes with Azure CLI, Azure PowerShell, and other tools pre-installed and ready to use.
  - **Persistent Storage**: Provides a persistent home directory in Azure Files, allowing you to save scripts and files across sessions.
- Could be set as Bash or PowerShell environment.

# Azure Arc
- Azure Arc is a set of technologies that extends Azure management and services to any infrastructure, including on-premises, multi-cloud, and edge environments. Key features include:
  - **Unified Management**: Manage resources across different environments from a single control plane in Azure.
  - **Azure Services Anywhere**: Deploy Azure services, such as Azure SQL Database and Azure Kubernetes Service, on any infrastructure.
  - **Governance and Compliance**: Apply Azure policies and security controls to resources outside of Azure.
- Can manage:
  - `Servers`
  - `Kubernetes clusters`
  - `Azure data services`
  - `SQL Server`
  - `Virtual machines (preview)`
