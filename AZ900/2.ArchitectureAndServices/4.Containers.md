# Azure Containers
- Containers are a lightweight, portable, and self-sufficient way to package and run applications. They allow developers to create, deploy, and manage applications consistently across different environments.

## Azure Container Instances (ACI)
- ACI is a serverless container service that allows you to run containers without managing the underlying infrastructure. 
- It provides a simple and fast way to run containers in Azure, with support for both Linux and Windows containers.

## Azure Container Apps
- Azure Container Apps is a fully managed serverless container service that enables you to run microservices and containerized applications without managing infrastructure. 
- It provides features such as automatic scaling, built-in Dapr support, and integration with Azure DevOps for continuous deployment.

## Azure Kubernetes Service (AKS)
- AKS is a managed Kubernetes service that simplifies the deployment, management, and operations of Kubernetes clusters in Azure. 
- It provides features such as automatic scaling, self-healing, and integrated monitoring to help you run containerized applications at scale.
